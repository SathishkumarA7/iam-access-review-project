# IAM Access Review Project — Q1 2026

## What this project is
A simulated quarterly Identity and Access Management (IAM) 
access review for a 15-person organisation — TechNova Pvt Ltd.

## What I built
- User Directory — 15 employees across IT, HR, Finance, Sales
- Access Matrix — system access inventory across 6 platforms
- Access Review — violation detection, risk rating, remediation actions
- Policy Document — formal findings report with recommendations

## Violations found and documented
- 3 Orphan Accounts (terminated employees with active access)
- 1 Privilege Violation (Junior Accountant with Admin Panel access)
- 1 Broken Manager Chain (active employee reporting to terminated manager)
- Overall Risk Rating: HIGH

## IAM concepts demonstrated
- Principle of Least Privilege
- Identity Governance and Administration (IGA)
- Access Certification / Access Recertification
- Orphan Account Detection
- Quarterly Access Review Process (SOC 2 / ISO 27001 aligned)

- Microsoft Word Online
